# VMCopy

Sends a file to a VM.

# Usage

```
vmcopy.exe --SourcePath "C:\Users\host\a.txt" --DestinationPath "C:\Users\guest\a.txt"
```

```
vmcopy.exe --SourcePath "%1"
```
